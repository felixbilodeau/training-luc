# Project Setup

1. Set python version

```
    pyenv local 3.8
```

2. Create virtual environment

```
    python -m venv venv
```

3. Activate environment

```
    source venv/bin/activate
```

4. Install Django

```
    pip install django
```


5. Apply migrations

```
    python manage.py migrate
```

6. Run server
```
    python manage.py runserver
```
